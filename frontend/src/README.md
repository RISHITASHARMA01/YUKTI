# Yukti Frontend

This directory contains the React frontend for the Yukti project.

## Prerequisites

You need to have Node.js installed on your computer to run this project.
Download it from: https://nodejs.org/

## Setup Instructions

1. Open a terminal in this directory (`frontend`).
2. Run the following command to install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the link shown in the terminal (usually http://localhost:5173).

## Features

- **Landing Page**: Complete migration of the original design to React components.
- **Donate Page**: Form to list food donations.
- **Receive Page**: Browse available donations.
- **Auth Page**: Login and Signup forms.
